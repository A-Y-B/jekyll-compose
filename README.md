# Jekyll::Compose

Streamline your writing in Jekyll with these commands.

## Installation

Add this line to your application's Gemfile:

    gem 'jekyll-compose'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jekyll-compose
